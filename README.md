# Consulta CNPJ

## Sobre o Projeto
  Projeto para simples consulta de CNPJ e facilitar os acessos a Cadastro de Fornecedores
  através da api pública "publica.cnpj.ws".
  Projeto sem estruturação Back-end, pois é uma página simples.

## Front-End
 - *Javascript*
 - *CSS*
 - *Jquery*
